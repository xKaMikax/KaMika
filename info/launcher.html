Описание проекта: Minecraft Node Launcher

Лаунчер для Minecraft на Node.js: оффлайн‑режим, кастомная Java, план поддержки Fabric/Quilt/Forge/NeoForge.

Цели

  • Запуск MC 1.21.1 с выбором Java 21 (javaw.exe).
  • Проверка/докачка ассетов и библиотек.
  • Поддержка модлоадеров (поэтапно).

CLI‑пример

node launcher.js --dir "D:\.minecraft" --version 1.21.1 --ram 4G --offline KaMikaYT --java "E:\Portable\java\bin\javaw.exe"

План работ

  • Fabric/Quilt — загрузка профиля, инсталляция, хук JVM аргументов.
  • Forge/NeoForge — поддержка инсталляторов, кеш библиотек.
  • GUI‑шаг: минимальное окно конфигурации.
